# Unity_TextureManager

<img src="https://github.com/XJINE/Unity_TextureManager/blob/master/Screenshot.png" width="100%" height="auto" />

``TextureManager`` provides simple system to manage some textures with Vector4 as meta.

## Import to Your Project

1. Import dependencies first.
2. Paste this code into ``PackageManager > Add package from git URL``.

```
https://github.com/XJINE/Unity_TextureManager.git?path=/Packages/TextureManager
```

### Dependencies

You have to import following assets to use this asset.

- [Unity_IInitializable](https://github.com/XJINE/Unity_IInitializable)
